LE WAGON PROJECT

Games master

Desktop application

Airbnb clone I did during Le Wagon Bootcamp. Games master allows the user to look for board games around him/her and to book them, or, on the other hand, to rent board games for the other users to play.

VISIT THE WEBSITE

http://games-master.herokuapp.com

GITHUB REPO

https://github.com/mauriarmora/rails-games-master

FEATURE

Categories

We created exactly 16 categories which you can choose from a big variety of games: adventure, classic, quiz, betting and many many more. Once you click on one of those, you will see all the games within that category and a map next to it, so you can see where are they located.

FEATURE

Searching
system

On the top of the main page there is a searching bar where you can type any board game that you want. If it is in our database, which I am sure it is, you will be able to see its location on the map. Ready to play? Go and search for your favorite board game!

FEATURE

Booking
a board game

You can either look for a game, or click on a game you would like to book. Then, you will be able to read a description about it and all the game specifications with a form next to it. There you can fill up the form with the dates when you would like to book the game. Once you do it, you can check your booking on your personal dashboard and wait for the owner's confirmation.
